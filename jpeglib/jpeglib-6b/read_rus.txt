    Для компиляции статических библиотек обьектных файлов IJG:
----------------------------------------------------------------------
        Visual C++ (v.6)
----------------------------------------------------------------------
- Переименовать файлы:
    а) jconfig.vc -> jconfig.h;
    б) makelib.ds -> makefile;
- Набрать в командной строке nmake.
- Скомпилированную библиотеку  "jpeg.lib", заголовочные файлы "jconfig.h", 
"jmorecfg.h" и  "jpeglib.h" скопировать в директорию проекта и использовать 
следующим образом:
    а) включить заголовочный файл
        extern "C" {
        #include "jpeglib.h"
        }
    б) включить библиотеку jpeg.lib в проект.

----------------------------------------------------------------------
        Builder C++ (v.5)
----------------------------------------------------------------------
- Переименовать файлы:
    а) jconfig.bcb5 -> jconfig.h;
    б) makelib.bcb5 -> makefile;
- Набрать в командной строке make.
- Скомпилированную библиотеку  "jpeglib.lib", заголовочные файлы "jconfig.h", 
"jmorecfg.h" и  "for_bcb/jpeglib.h" скопировать в директорию проекта и 
использовать следующим образом:
    а) включить заголовочный файл
        extern "C" {
        #include "jpeglib.h"
        }
    б) включить библиотеку jpeglib.lib в проект.роект.
